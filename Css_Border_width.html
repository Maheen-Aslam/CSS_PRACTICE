Css Border Width

The border-width property specifies the width of the four borders. The width can be set as a specific size (in px, pt, cm, em, etc) or by using one of the three pre-defined values: thin, medium, or thick:

Thin(Border:2px Style:solid)

Medium(Border:medium Style:solid)

Thick(Border:thick Style:Dotted)

Specific Side Width

The border-width property can have from one to four values (for the top border, right border, bottom border, and the left border):

Examples

1.Top Or Bottom 5px left or right 20px (BorderStyle:Solid)

2.Top or Bottom 20px left or right 5px(BorderStyle:Solid)

3. 25px top, 10px right, 4px bottom and 35px left(BorderStyle:Solid)
